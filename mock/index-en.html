What is blockchain games(DAPPS)?

Blockchain game is a game which choose blockchain as database. It has some pros, data permanency(data is not deleted when game service is finished.) and data transparency(player becomes not necessary to suspect fraud.) etc.

Strength of this game

Even if the data is persist after DAPPS service ended, It is no sense without place to use the data. This game is the place to use image even if some DAPPS service ended, and the platform for creator who sell the card with his own picture.
